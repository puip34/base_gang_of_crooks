Get
Endpoint: /users/
Описание: Получение информации обо всех пользователях системы.

Create
Endpoint: /users/
Описание: Создайте нового пользователя в системе.

Update
Endpoint: /users/{user_id}
Описание: Обновление информации для существующего пользователя.

Delete
Endpoint: /users/{user_id}
Описание: Удаление пользователя на основе его идентификатора пользователя.
